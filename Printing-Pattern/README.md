 
# Java_programs

This repository contains Java programs related to printing patterns.

## Star1

### Square Pattern

**Output:**


*PS D:\Java\Java_programs\Printing-Pattern> javac star1.java*

*PS D:\Java\Java_programs\Printing-Pattern> java star1*
```  
* * * * * * 
* * * * * *
* * * * * *
* * * * * *
* * * * * * 
* * * * * *
```
